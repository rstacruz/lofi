# lofi

> Lofi player

## Developer setup

```bash
yarn
yarn run develop
```

## Other commands

| Command          | Description                                            |
| ---------------- | ------------------------------------------------------ |
| `yarn develop`   | Starts development server (defaults to localhost:8000) |
| `yarn test`      | Run tests                                              |
| `yarn storybook` | Open Storybook (defaults to localhost:9001)            |
